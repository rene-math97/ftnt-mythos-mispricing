## Thesis summary

### Setup
On March 27, 2026, Anthropic released the "Claude Mythos" capability demonstration, which included cybersecurity agent workflows. The cybersecurity cohort sold off indiscriminately over the following days. FTNT fell with the cohort despite having the lowest AI-displacement risk profile in the group (network-layer hardware + software moat, not endpoint/SaaS).

### Variant perception
The market is treating FTNT like a high-multiple SaaS name exposed to AI displacement. It isn't:
- **8.2x EV/Revenue** vs. 12.3x peer median — already trading at a discount
- **36.8% FCF margin** — highest in cybersecurity
- **31% GAAP operating margin** — only GAAP-profitable name in the cohort
- **755,000 customers**, **$6.8B revenue** — diversified, sticky installed base
- **Largest firewall vendor in the world** — moat is silicon (custom ASICs), not software features

### Why not CrowdStrike?
Original research direction was long CRWD. A full DCF across bull/base/bear scenarios showed CRWD was overvalued in every case — pivoted to FTNT when the math said so. The web app includes a dedicated "Why Not CrowdStrike?" section walking through that comparison.

### Asymmetric payoff
- Bull case: +45%
- Base case: +28.5%
- Bear case: +12% (still positive — downside is cushioned by FCF yield and buyback authorization)
